# primetrade-trader-sentiment-analysis
# Trader Performance vs Market Sentiment Analysis

## Objective
This project analyzes how Bitcoin market sentiment (Fear vs Greed) affects trader behavior and profitability on Hyperliquid.

## Datasets
- Fear/Greed Index dataset
- Historical Hyperliquid trader dataset

## Key Metrics Created
- Daily PnL per trader
- Win rate
- Trade frequency
- Long/Short ratio
- Leverage behavior

## Insights
- Performance differs between Fear and Greed days
- Traders change leverage and activity depending on sentiment
- High leverage traders are more sensitive during Fear periods

## Strategy Recommendations
- Reduce leverage exposure during Fear sentiment
- Increase activity only for consistent profitable traders during Greed days
